# Claude Code for Illustrator

**イラストレーターのためのClaude Code完全構築テンプレート**

このリポジトリにある `ClaudeCode_illustrator_setup.md` をClaude Codeに渡すだけで、AIがあなたの業務をヒアリングしながら、イラストレーター向けのAI業務管理システムをゼロから自動構築してくれます。

---

## 🎁 何ができるようになるか

- 朝「おはよう」と入力するだけで、本日の工程表が自動生成
- ストック1枚 → 多言語キーワード50個を1分で自動生成
- クライアントメール → 返信ドラフトを5分で生成
- 月次収支を自動集計・レポート
- X / Threads / Instagram の投稿下書きを自動生成
- イラスト企画 → ラフ案3パターンをAI提案
- 見積もり・請求書を自動ドラフト化

体感で**月40時間**の業務が自動化できます。

---

## 🚀 使い方（3ステップ）

### Step 1: Claude Code をインストール

[公式サイト](https://claude.com/claude-code)からダウンロード。Mac / Windows 対応。Claude Pro プラン（$20/月）が必要です。

### Step 2: このリポジトリのテンプレをダウンロード

このページの `ClaudeCode_illustrator_setup.md` を開いて、右上の「Raw」ボタンから全文コピー。

または以下のURLから直接アクセス：
```
https://raw.githubusercontent.com/fjsw8376-dotcom/claude-code-for-illustrator/main/ClaudeCode_illustrator_setup.md
```

### Step 3: Claude Code に貼り付ける

Claude Code を起動して、コピーしたテキストを丸ごと貼り付けて Enter。

あとはAIが順番に質問してくるので、答えていくだけ。10分後にはあなた専用の業務管理システムが完成しています。
